Demo code for "simple lossless index buffer compression".

This uses the "Armadillo" model from the Stanford 3D scanning repository:

  http://graphics.stanford.edu/data/3Dscanrep/

use the included "ply_import.py" script to convert the .ply file from the
web site to a .bin file the script understands.
